UIkit 2 - A lightweight and modular front-end framework.

== Description ==

The official description of the UIkit framework:
"A lightweight and modular front-end framework for developing fast and powerful web interfaces."

This plugin does only one thing: it enques all the necessary css and js files for you to use the UIkit framework. The css and js files are minified already to make it as light as possible.

UIkit theme: Default

UIkit Comonents included with this plugin:
- Button
- Dropdown
- Icon
- Nav
- Panel
- Switcher
- Tab
- Table
- Utility

For more info about UIkit and all the components:
- [UIkit](http://getuikit.com/)

== Installation ==

1. Upload the `surbma-uikit-2-default` folder to the `/wp-content/plugins/` directory
2. Activate the Surbma - UIkit 2 - Default plugin through the 'Plugins' menu in WordPress
3. That's it. You can now use UIkit components on your site. (Not all of them!)
